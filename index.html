Algorithmic Trading

Prof. Weiqing Gu
Summer 2020

Course Description

In this course, we cover Bayesian statistics, Bayesian inference of a binomial proportion, Markov chain Monte Carlo, Bayesian linear regression, time series analysis, serial correlation, random walks and white noise models, autoregressive moving average models, autoregressive integrated moving average and conditional heteroskedastic models, state space models and the Kalman filter, statistical machine learning, kernel methods and SVMs, quantitative trading strategies, and ARIMA+GARCH trading strategy on stock market indexes.

You will learn how to read an academic paper with available code in some github and give an summary of the paper and reproduce the results using the code in the GitHub.

Since algorithmic trading is a method of executing orders using automated pre-programmed trading instructions accounting for variables such as time, price, and volume, we will learn how to teach the machine to the trading by making use of the speed and data processing advantages that computers have over human traders.

Summary of Goals

  • Gain a comprehensive view of trading algorithms and trading technologies and understand the mathematics behind it.

  • Be able to search and read recent academic papers in the algorithmic trading literature and apply those algorithms and concepts to real world problems.

  • Become comfortable with industry and academia standard tools (such as AWS and GitHub) and be able to find and work with large, public datasets.
